What is the Large KB gazetteer

Well, it's easy enough to explain. The large KB gazetteer allows loading huge collections of identifiers and labels and using them for gazetteer lookup. This gazetteer extracts can dictionary from a remote RDF repository, local set of RDF files ot pregenerated binary image of a such collection.
