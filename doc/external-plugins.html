Multi-lingual Noun Phrase Extractor (NPE)
NPE NPE is a multi-lingual noun phrase (NP) extraction component developed for the GATE architecture, implemented in JAPE. It currently supports English, German, and French. To download and for more information, see http://www.ipd.uka.de/~durm/tm/npe/ en-np_main.jape
